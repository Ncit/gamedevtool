dialog-authoring
SKILL
Build a branching conversation as a dialog node that owns dialog_line nodes, with player choices wired as typed edges and a speaker attached.
WHEN TO USE
The player talks to an NPC and their choices branch the conversation or set flags used elsewhere in the design.
STEPS
1
Create a dialog node, owned by the speaking NPC.
2
Add a dialog_line child for each beat of the conversation.
3
Wire player choices as branches_to edges, each carrying a condition.
4
Link the speaker with a speaks edge from the NPC to the dialog.
5
Keep every line reachable — no orphan lines.
VERIFY
find_dialog_dead_ends()
